# 视频管理系统

一个基于 React 和 Trickle 数据库的视频管理系统，支持用户注册、登录、视频上传和管理员后台功能。

## 功能特性

### 用户功能
- **用户注册/登录**：支持用户名、四位学号和密码登录，首次登录自动创建账户
- **视频上传**：用户可以从电脑本地选择并上传视频文件，支持所有常见视频格式
- **视频管理**：查看自己上传的所有视频，带上传进度显示

### 管理员功能
- **管理员登录**：使用特殊账户（admin/0000/adminstrator）登录
- **用户管理**：查看所有注册用户列表
- **视频监控**：查看每个用户上传的视频

## 页面结构

- `index.html` - 登录页面
- `home.html` - 用户主页（视频列表和上传）
- `admin.html` - 管理员后台

## 数据库表结构

### user 表
- username: 用户名
- studentId: 四位学号
- password: 密码

### video 表
- userId: 用户ID
- username: 用户名
- studentId: 学号
- title: 视频标题
- url: 视频链接
- uploadDate: 上传日期

## 使用说明

1. 访问登录页面，输入用户名、学号（四位）和密码
2. 首次登录会自动创建账户
3. 普通用户登录后进入主页，可以上传和查看视频
4. 管理员登录后进入后台，可以查看所有用户及其视频

## 管理员账户

- 用户名：admin
- 学号：0000
- 密码：adminstrator
